# 🩺 Public Health Analytics Dashboard  
### 📊 Diabetes Predictor – Patient Health Analysis

## 📌 Project Overview  
This project is a Public Health Analytics Dashboard built using Python and Streamlit.  
It analyzes patient health data and predicts the risk of diabetes.

---

## 🚀 Features  
- Total Patient Analysis  
- Average Sugar Level  
- High Risk Detection  
- Data Visualization (Graphs & Charts)  
- Diabetes Prediction Model  
- Model Accuracy: 100%  

---

## 🛠️ Technologies Used  
- Python  
- Streamlit  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 📂 Files Included  
- app.py  
- dataset.csv  
- README.md  

---

## ▶️ How to Run  

```bash
pip install streamlit pandas numpy matplotlib scikit-learn
streamlit run app.py
```
